Walliya UNICEF Final Project

How to run:
1. Create a virtual environment (recommended)
   python -m venv .venv
   source .venv/bin/activate (Linux/Mac)
   .venv\Scripts\activate (Windows)

2. Install requirements:
   pip install -r requirements.txt

3. Open 'Unicef-Quarto-Walliya.qmd' in VS Code.

4. Render:
   quarto render Unicef-Quarto-Walliya.qmd

Output:
- Beautiful HTML with 4 graphs (map, bar chart, line chart, scatter plot)
